Component Structure

Customer Portal Components (Mobile Wallet Style)

  • Layout/App.jsx - Main app wrapper with bottom navigation
  • Loading/LoadingScreen.jsx - Loading screen with orange background and white text
  • Home/WalletCard.jsx - Flip-able wallet card (front: balance, back: QR code)
  • Home/QRCodeView.jsx - Full QR code view screen (shown when tapping card or Show QR button)
  • Home/QuickActions.jsx - Quick action buttons (Show QR, Rewards, History)
  • Home/ProgressCard.jsx - Progress to next card design unlock
  • Home/PromotionBanner.jsx - Promotional banner carousel
  • Rewards/RewardsList.jsx - List of available rewards
  • Rewards/RewardCard.jsx - Wallet-style reward card with image
  • Rewards/ClaimModal.jsx - Claim confirmation modal with voucher
  • Collections/CardDesignsGrid.jsx - Grid of card design collectibles
  • Collections/CardDesignCard.jsx - Individual card design preview (locked/unlocked states)
  • Collections/ActiveCardPreview.jsx - Preview of currently active card design
  • Promotions/BannerCarousel.jsx - Promotional banner carousel
  • Profile/ProfileCard.jsx - Wallet-style profile card
  • Profile/StatsCards.jsx - Current and lifetime espro coins cards
  • ClaimHistory/ClaimHistoryList.jsx - List of claimed rewards
  • ClaimHistory/ClaimedRewardCard.jsx - Individual claimed reward card
  • ClaimHistory/VoucherModal.jsx - Modal to view voucher code and details
  • Navigation/BottomNav.jsx - Fixed bottom navigation bar
Wallet Design Principles:
  • Card-based UI with gradient backgrounds
  • Flip animation for wallet card (balance QR code)
  • Premium feel with subtle shadows and rounded corners
  • Transaction-style history view
  • Quick actions for common tasks
  • Clean, minimal design with brand colors

Typography & Design Guidelines

Typography

  • Font Family: System fonts (-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, etc.)
  • Headings: Bold, using brand colors (#4b2e2b for primary, #f66633 for accents)
  • Body: Regular weight, #333333 for primary text, #666 for secondary
  • Monospace: For loyalty IDs and codes

Spacing & Layout

  • Mobile: 16px padding, 8px gaps between elements
  • Desktop: 24px padding, 16px gaps
  • Cards: 12px border radius, subtle shadows
  • Buttons: 8px border radius, primary color background

Interactions

  • Hover: Subtle opacity change (0.9) or background color shift
  • Active: Slight scale (0.98) for touch feedback
  • Transitions: 150ms ease for all interactions
  • No black highlights: All tap highlights removed
